# Describe

Inspect models and field definitions.

The docs surface maps to the model-first API: list models, inspect a single
model, or work inside its `fields` branch.

Common uses:

- `abbot api describe list`
- `abbot api describe get users`
- `abbot api describe fields list users`
- `abbot api describe fields get users name`

Use `--help` on `list`, `get`, or `fields` for the next level down.
