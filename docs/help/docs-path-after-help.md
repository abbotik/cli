# Docs Path

Fetch a docs page by path.

Useful paths include:

- `/docs`
- `/docs/auth`
- `/docs/api/keys`
- `/docs/api/user/machine-keys`
- `/docs/api/data`
- `/docs/api/describe`
- `/docs/api/find`
- `/docs/llm/room`
- `/docs/llm/factory`
- `/docs/api/cron`

This branch is intentionally concise; use the command itself for the full flag list.
