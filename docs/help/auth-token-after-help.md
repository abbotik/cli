# Auth Token

Inspect or manage the saved JWT stored in the local Abbotik config.

Use this when you need to view the current session token, overwrite it
manually, or clear it from disk.

Common uses:

- `abbot auth token get`
- `abbot auth token set @token.txt`
- `abbot auth token clear`

Use `--help` on `get`, `set`, or `clear` for the next level down.
