# Public

Open the API discovery pages that are safe to read without auth.

By default these requests go to `https://api.abbotik.com` unless you override the
base URL.

Good starting points:

- `abbot public root` for the human-facing discovery page
- `abbot public llms` for the agent-facing markdown entrypoint

Use `--help` on `root` or `llms` for the next level down.
